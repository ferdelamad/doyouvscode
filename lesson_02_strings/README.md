# Lesson 02 - Strings

### Part I

Commands used in this lesson:

| `Keybinding` | `Description`                 |
| ------------ | ------------------------------|
| ⇧ ⌥ ↑        | Duplicate line up             |
| ⇧ ⌥ ↓        | Duplicate line down           |
| ⌥ ↑          | Move line up                  |
| ⌥ ↓          | Move line down                |
| ⌘ X          | Cut line                      |
| ⇧ ⌘ K        | Delete line                   |
| fn ⇧ →       | Select line from start to end |
| fn ⇧ ←       | Select line from end to start |

### Part II

Commands used in this lesson:

| `Keybinding` | `Description`                               |
| ------------ | --------------------------------------------|
| ⇧ ⌘ L        | Select all occurrences of current selection |
| ⌘ D          | Add next match to selection                 |
| ⌘ G          | Find next                                   |
| ⇧ ⌘ G        | Find previous                               |
