# Lesson 04 - Lines

Commands used in this lesson:

| `Keybinding` | `Description`                 |
| ------------ | ------------------------------|
| ⇧ ⌥ ↑        | Duplicate line up             |
| ⇧ ⌥ ↓        | Duplicate line down           |
| ⌥ ↑          | Move line up                  |
| ⌥ ↓          | Move line down                |
| ⌘ X          | Cut line                      |
| ⇧ ⌘ K        | Delete line                   |
| fn ⇧ →       | Select line from start to end |
| fn ⇧ ←       | Select line from end to start |
