# Lesson 04 - Strings

Commands used in this lesson:

| `Keybinding` | `Description`                               |
| ------------ | --------------------------------------------|
| ⇧ ⌘ L        | Select all occurrences of current selection |
| ⌘ D          | Add next match to selection                 |
| ⌘ G          | Find next                                   |
| ⇧ ⌘ G        | Find previous                               |
